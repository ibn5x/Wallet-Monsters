This folder contains all the source files used to design and export the assets.
The directory structure is as follows:
- Animation: Dragonbones source files (For more information check out Dragonbones website: http://dragonbones.com)
- Background And Environment: AI and EPS source files
- Characters: AI and EPS source files
- Enemies: AI and EPS source files
- Items: AI and EPS source files
- UI: AI and EPS source files
