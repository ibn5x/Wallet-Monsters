This folder contains the exported textures and images.
The directory structure is as follows:
- Background And Environment: Both sheets and images are available in 2 sizes (Clouds are available in 3 sizes)
- Characters: Sheets are available in 3 sizes
- Enemies: Images are available in 3 sizes
- Tiles: Images are available in 3 sizes
- UI
-- Button: Both sheets and images are available in 3 sizes
--- Empty and Gray buttons are included for custom color and styling (Or you can use it as disabled state)
-- Icons: Images are available in 3 sizes
-- Windows: Images are available in 3 sizes
